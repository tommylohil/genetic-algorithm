# Genetic Algorithm
Genetic Algorithm Implementation in java

## Prerequisite
Install Maven https://maven.apache.org/install.html

## How to run
```
mvn package assembly:single
java -jar target/genetic-algorithm-app-1.0-SNAPSHOT-jar-with-dependencies.jar
```
